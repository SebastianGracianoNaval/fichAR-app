# fichAR macOS

## When to Use

- macOS build and signing
- Notarization for distribution
- Permissions (camera, location, etc.)
- Sandbox configuration

## Compatibility

- macOS 10.14+ (Mojave)

## Distribution

- Notarization required for Gatekeeper
- Code signing with Apple Developer ID
