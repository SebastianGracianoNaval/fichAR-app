# fichAR Regression Check

## When to Use

- Before merging a bug fix or feature change
- When change touches shared code

## Output

1. **Affected areas**: Modules, flows, screens
2. **Tests to run**: Specific test commands
3. **Edge cases**: CASOS-LIMITE that may be impacted
4. **Manual checks**: If any
